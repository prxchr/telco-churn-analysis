Telco Customer Churn Analysis

Objective
Analyze customer churn behavior in a telecom company and identify key factors influencing churn.

Dataset
- Source: [Kaggle - Telco Customer Churn](https://www.kaggle.com/datasets/blastchar/telco-customer-churn)

Tools Used
- Python (Pandas, Matplotlib, Seaborn, Scikit-learn)
- Google Colab
- GitHub

Key Insights
- Month-to-month contracts have the highest churn rate.
- Higher monthly charges and lower tenure are linked to higher churn.
- Customers with fiber optic internet and multiple services are more likely to churn.
